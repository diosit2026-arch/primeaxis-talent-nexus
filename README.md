# Primeaxis HR Solutions

Primeaxis HR Solutions is a Vite + React + TypeScript website for showcasing recruitment, staffing, payroll, compliance, and HR consulting services.

## Scripts

- `npm run dev` starts the local development server
- `npm run build` creates a production build
- `npm run preview` previews the production build locally
- `npm run test` runs the test suite

## Netlify Deployment

- Build command: `npm run build`
- Publish directory: `dist`
- SPA routing is handled with `netlify.toml`

## Pages

- `/` Home
- `/about` About
- `/services` Services
- `/contact` Contact
